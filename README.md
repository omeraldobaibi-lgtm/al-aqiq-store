# مشروع متجر العقيق - جاهز للنشر الخارجي

## مهم
لا يوجد ملف واحد يمكنه أن يجعل الموقع متاحًا من شبكة خارجية بدون استضافة أو سيرفر عام.
هذا المشروع **جاهز للنشر** على Render أو أي استضافة Node.js.

## التشغيل المحلي
```bash
npm install
npm start
```

## النشر على Render
1. ارفع المشروع إلى GitHub
2. في Render اختر New > Blueprint أو Web Service
3. إذا استخدمت Blueprint فسيقرأ ملف render.yaml تلقائيًا
4. بعد النشر سيعطيك رابطًا عامًا مثل:
   https://your-app.onrender.com

## روابط المشروع محليًا
- المتجر: http://localhost:3000
- الإدارة: http://localhost:3000/admin

## كلمة مرور الإدارة
admin123

## ملاحظة
الطلبات محفوظة في ملف JSON محلي، وهذا مناسب للتجربة والبداية.
في الاستضافة السحابية قد تحتاج قاعدة بيانات لاحقًا للحفظ الدائم.
